**components/sections/HeroLight.tsx** — Shared light-background hero for Home, Services, About, Contact, Portfolio:

Props interface:
```typescript
interface HeroLightProps {
  kicker?: string;           // Caption kicker, e.g., "SERVICES"
  headline: string;          // Display text
  headlineSize?: "text-display-xl" | "text-display-lg" | "text-display-md";
  highlightWord?: string;    // Word to wrap in accent span (e.g., "engineered")
  intro?: string;            // Body-lg intro paragraph
  primaryCTA?: { label: string; href: string };
  secondaryCTA?: { label: string; href: string };
  showScrollCue?: boolean;   // Show "scroll" text with line animation (home only)
  className?: string;
  children?: React.ReactNode; // For additional content (e.g., WebGL placeholder)
}
```

Implementation:
- Section padding: `py-5xl` desktop (192px), `py-3xl` mobile (72px). Full viewport height for home hero (`min-h-[calc(100vh-64px)]` when headlineSize is display-xl), shorter for other pages.
- Container: default size (1200px)
- Kicker: `<Caption>` component if provided
- Headline: render as `<h1>` (not h2 — this is the page heading). Apply the headlineSize class. If `highlightWord` is provided, find that word in the headline and wrap it in `<span className="text-accent">{highlightWord}</span>`.
- Intro: `<p className="text-body-lg text-text-muted mt-lg max-w-[640px]">{intro}</p>`
- CTAs: flex row with gap-md, primary Button + secondary Button (if provided). `mt-2xl`.
- Scroll cue (if showScrollCue): absolute bottom of section, centered. `<span className="text-mono-sm text-text-muted">scroll</span>` with a 1px vertical line below it, 32px tall, using a `<div className="w-px h-[32px] bg-border-light mx-auto mt-sm">` (the line animation will be added in Phase 3).
- Children rendered after CTAs (for WebGL placeholder slot).
- Server Component.
